# Neo Rose Theme for OpenCode

A dark neo-terminal theme with rose, cyan, and violet accents on deep void background.

## Installation

Add to your OpenCode configuration:

```json
{
  "theme": "neo-rose"
}
```

## Colors

- **Primary**: Neon pink (#ff4da6)
- **Secondary**: Cyan neon (#00f0ff)
- **Accent**: Violet neon (#bd34fe)
- **Background**: Deep void (#0a0812)

## Companion Plugin

For the full neo-terminal experience with scanlines, vignette, and neural effects, install the companion plugin:

```json
{
  "theme": "neo-rose",
  "plugin": ["~/.config/opencode/oc-neo-terminal-dev/packages/plugin"]
}
```
