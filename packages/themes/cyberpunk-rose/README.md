# Cyberpunk Rose Theme for OpenCode

A dark cyberpunk theme with rose/magenta accents for OpenCode.

## Installation

Add to your OpenCode configuration:

```json
{
  "plugin": ["oc-theme-cyberpunk-rose"]
}
```

## Colors

- **Primary**: Neon pink (#ff4da6)
- **Secondary**: Cyan neon (#00f0ff)
- **Accent**: Violet neon (#bd34fe)
- **Background**: Deep void (#0a0812)

## Companion Plugin

For the full cyberpunk experience with scanlines, vignette, and neural effects, install the companion plugin:

```json
{
  "plugin": ["oc-theme-cyberpunk-rose", "oc-plugin-cyberpunk"]
}
```
